Cores Predefinidas

Os browsers suportam 140 cores pré definidas. Ex: color: red;

RGB

RGB representa Red, Green e Blue rgb(255,0,0);

RGBA

Já o RGBA adiciona um nível de transparência rgba(255,255,18,0.1);

Hexadecimal

Já o Hexadecimal é composto pelo sinal # os dois primeiros caracteres representam a intensidade da cor vermelha os dois do meio representam a cor verde e os dois últimos a cor azul. EX: #f1f1f1

Para adicionar transparência no hexadecimal só adicionar 2 digitos ao final com valores entre 00 e ff EX: #0000FF3F

HSL

Hue(matiz) é o grau na roda de cores (de 0 a 360 graus)

  • 0 ou 360 é vermelho
  • 120 é verde
  • 240 é azul

Saturation(saturação) valor percentual

  • 0% é preto
  • 100% é branco

Lightness(luminosidade) luminosidade da cor

  • 0% é preto
  • 100% é branco

EX: color: hsl(45,90%,20%);

Já com o HSLA é possível colocar um nível de transparência que aceia valores de 0.0 até 1. EX: hsla(45,100%,50%,0.5);
